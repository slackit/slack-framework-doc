Slack App Setup

  1. Before starting any implementation, You are required to define your Slack credentials by implementing the following Contract:

    It's recommended that all of credentials should be stored in a seperate place from the source code. For example, you can store the credentials in database or you can retrieve them from .env file.

    
<?php

namespace Slackit\LaravelSlackSDK\Contracts;

interface AppInformationContract
{
    /**
     * @return string
     */
    public function getAppId(): string;

    /**
     * @return string
     */
    public function getAppName(): string;

    /**
     * @return string
     */
    public function getAppVersion(): string;

    /**
     * @return string
     */
    public function getClientId(): string;

    /**
     * @return array
     */
    public function getAppScopes(): array;

    /**
     * @return string
     */
    public function getClientSecret(): string;

    /**
     * @return string
     */
    public function getSigningSecret(): string;

    /**
     * @return string
     */
    public function getRedirectUrl(): string;
}
