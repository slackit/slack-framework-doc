Single Command - Hello World!

Hello World Example

  1. Create a Hello World command class that extends AbstractSlackCommand, see below:

    This is just a simple command example that helps you understand the abstract class.

    
<?php

use Slackit\LaravelSlackSDK\Services\Core\Command\Contracts\AbstractSlackCommand;

/**
 * Class SimpleCommand
 * @package App\Slack\Commands
 */
class HelloWorldSlackCommand extends AbstractSlackCommand
{
    /**
     * Check if the user has authority to access the command
     * @return bool
     */
    public function authorize()
    {
        return true;
    }

    /**
     * Your extra command validation logic
     * @return bool
     */
    public function validate()
    {
        return true;
    }

    /**
     * @return array|mixed
     */
    public function process()
    {
        return 'Hello World!';
    }
}

    You can manually set authorize() and validate() to return false if you want to see the sample message.

    In next chapters, you will learn how to customize these messages as well as the message from process().

  1. Register your command in Controller:

    
<?php


use App\Http\Controllers\Controller;

class SlackHelloWorldController extends Controller
{

    /**
     * @param HelloWorldSlackCommand $helloWorldSlackCommand
     * @return mixed|string
     */
    public function helloWorld(HelloWorldSlackCommand $helloWorldSlackCommand)
    {
        return $helloWorldSlackCommand->execute();
    }
}

  2. Register a route to call the above Controller function.

  3. Done! Now you just need to create a new command in Slack and trigger the command in Slack chat.
