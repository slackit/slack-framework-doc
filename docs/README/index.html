Laravel Slack SDK

A complete Slack SDK. Unlike other packages, this library provides Slack features as many as possible, everything comes in out of the box.

Planned release date: 31/08/2019

(Private) Pre-installation

  1. You have already set up the main project.

(Private) Installation

This is a private repo, so the installation would be a bit different from public package.

We need to set up a local Laravel package development environment.

  1. Install a helper package that quickly helps us set up a structure of package development in Laravel:
composer require jeroen-g/laravel-packager --dev
  1. Run the following artisan command to generate a folder structure for package development:
php artisan packager:new TimeHunter LaravelSlackSDK --i

Skip all questions.

You should see a new packages folder created under project directory.

  1. Go to packages folder and go to TimeHunter folder, delete the whole folder LaravelSlackSDK
  2. Git clone the project: git clone https://github.com/RyanDaDeng/laravel-slack-sdk.git LaravelSlackSDK
  3. Add /packages into your project .gitignore file

Set up

run:

php artisan vendor:publish --tag=slacksdk.config --force

Usage

Command

Single Command

use TimeHunter\LaravelSlackSDK\Services\Command\Contracts\AbstractSlackCommand;
use TimeHunter\LaravelSlackSDK\Services\Core\Messages\SlackAttachment;
use TimeHunter\LaravelSlackSDK\Services\Core\Messages\SlackMessage;

/**
 * Class SimpleCommand
 * @package App\Slack\Commands
 */
class SimpleCommand extends AbstractSlackCommand
{
    /**
     * @var string $description
     */
    protected $description = 'Hello Slack - command description';

    /**
     * Your extra command validation logic
     * @return bool
     */
    public function validate()
    {
        return true;
    }

    /**
     * @return mixed|void
     */
    public function handle()
    {
        // Respond to slack command payload response_url
        $this->respondToSlack('Hello, Slack!');

        // An instant response
        return SlackMessage::create()->withAttachment(SlackAttachment::create()
            ->setInfoColor()
            ->setText('Greeting payload message')
            ->setFallback('Greeting payload message')
        )->toArray();
    }
}

Group Command

use TimeHunter\LaravelSlackSDK\Services\Command\SlackGroupCommandProcessor;

/**
 * Class SimpleGroupCommand
 * @package App\Slack\Commands
 */
class SimpleGroupCommand extends SlackGroupCommandProcessor
{
    /**
     * @var array $commands
     */
    protected $commands = [
        'hello' => SimpleCommand::class
    ];

}

    protected $blockActions = [
        LeaveApprovedAction::class,
        LeaveDeclinedAction::class,
        LeaveRedoAction::class
    ];
}
