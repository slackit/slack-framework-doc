Payload

Background

Slack has a mechanism to protect their request by verifying their payload signature. This is a MUST requirement that any application should implement it.

Usage

  1. Firstly, make sure you have already implemented AppInformationContract which has been mentioned in Slack App Setup

  2. Copy the following code in your Controller class. This is just a standard Laravel middleware class.

    The class here HelloWorldAppInfo is a fake class which implements AppInformationContract. You should replace it with your own class.

        /**
     * SlackCommandController constructor.
     * @param LeaveBotAppInfo $leaveBotAppInfo
     */
    public function __construct(HelloWorldAppInfo $helloWorldAppInfo)
    {
        // verify command request from Slack
        $this->middleware(VerifySlackSignature::class . ':' . $helloWorldAppInfo->getSigningSecret());
    }

    You can define the middleware VerifySlackSignature::class anywhere as long as it intercepts your controller functions.
