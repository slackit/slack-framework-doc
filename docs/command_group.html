Group Command

Make sure you have completed Slack Command - Hello World section before start reading this section.

What is Group Command?

The Group Command is a collection of single commands that you can define as many as you want.

For example, you want to have more than 10 slack commands. In traditional way, you are required to create 10+ commands in Slack website which you may find its time consuming. The end user may experience hard time to remember all the commands.

The Group Command is a solution that help you combine all commands into just one Slack Command.

Sample Use Case

Let's assume we want to implement the following commands:

- Hello World command which prints 'Hello World!'
- Hello Slack command which prints 'Hello Slack!'

If we implement them by using Single Command, we may have two commands defined as below:

/hello-world
/hello-slack

What if we want to call them via just one Slack Command as like:

/hello-bot

and the usage could be:

/hello-bot world
/hello-bot slack

Yes, this is what exactly the Group Command can help you achieve it.

Create Group Command

  1. Firstly, let's define two more properties in the Hello World command class:

        /**
     * Class SimpleCommand
     * @package App\Slack\Commands
     */
    class HelloWorldSlackCommand extends AbstractSlackCommand
    {
       ...
        /**
         * @var string $description
         */
        protected $description = 'print Hello World';
    
        /**
         * @var string $name
         */
        protected $commandName = 'world';
       ...
    }

    $commandName and $description are used to define your command signature and details.

    It's important to ensure that $commandName is always unique. This property is a map key that used to let system know what command is being triggered.

  2. Create your Group Command class:

        use Slackit\LaravelSlackSDK\Services\Core\Command\SlackGroupCommandProcessor;
    
    class HelloBotGroupCommand extends SlackGroupCommandProcessor
    {
        /**
         * Define your commands
         * @var array $commands
         */
        protected $commands = [
            HelloWorldCommand::class,
            HelloSlackCommand::class,
        ];
     
        /**
         * Check if the user can access the command
         * @return bool
         */
        protected function authorize()
        {
            return true;
        }
        
    }

  3. Create a controller function

    
 /**
  * @param HelloBotGroupCommand $helloBotGroupCommand
  * @return mixed|string
  */
 public function helloBot(HelloBotGroupCommand $helloBotGroupCommand)
 {
     return $helloBotGroupCommand->execute();
 }

  4. Register a route.

  5. Done!

:) Try to call /hello-bot, /hello-bot world and /hello-bot slack to see what the result is.
